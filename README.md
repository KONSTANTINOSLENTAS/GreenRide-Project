Οδηγίες Εγκατάστασης και Εκτέλεσης (README)

5.1 Προαπαιτούμενα Λογισμικού
Java 17 (ή νεότερη)
Node.js (v18+) & npm
Maven (για το build του Backend)




5.2 Εκτέλεση Backend (Server)
Πλοηγηθείτε στον φάκελο της εργασίας: cd greenride-backend
Εκτελέστε την εντολή: mvn spring-boot:run
Ο server θα τρέχει στη διεύθυνση: http://localhost:8080
Η τεκμηρίωση του API θα είναι διαθέσιμη στο: http://localhost:8080/swagger-ui.html





	5.3 Εκτέλεση Frontend (Client)
Πλοηγηθείτε στον φάκελο: cd greenride-frontend
Εγκαταστήστε  npm install
Εκκινήστε την εφαρμογή: npm start
Η εφαρμογή θα ανοίξει αυτόματα στον browser: http://localhost:3000
